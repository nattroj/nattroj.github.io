Intro

Hello, I'm Nattajohn (Nate) Rojanasupya! I am currently a student at Lambda School in the WEB24 cohort. Like many of us in this track, I am here to hone my skills as an up and coming fullstack web developer. By the way, check out my awesome work.

Work

Recent Projects:


Scheduler

The scheduler is an applet with a basic UI made with PySimpleGUI. This applet allows the scheduling department to query data from the local database, retrieve new data from the gas company's database, and interact with the G Suite Calendar using Google's API in one simple interface.

Mobile Forms

The Mobile Forms solution my employer opted for used Javascript as it's backend language for providing logic to the digital forms. After stretching the limits of this software, it is now being deployed and used by over 50 of our field technicians and errors are at an all time low! Because the forms are done in Javascript, I was able to retrieve all the field data and convert to JSON in order to store it in our production MongoDB.

Automated Invoicing

Using a Python script that takes advantage of the requests and lxml libraries, my employer is now able to automate data entry to the local gas company's website. This is done using GET and POST requests to scrape the tokens off of the html and POST back the data we want to enter into JSON format (data is pulled from MongoDB) as form data

About

My first exposure to programming happened when my employer made the decision to go paperless. I had two options:

  • Watch them hire some guy at the Mobile Forms company to convert all of our physical forms to digital forms and add logic for $100.00/hr (Javascript).
  • BECOME THAT GUY!

This was certainly not the hardest choice I've made in my life and it has lead me down a path of no return. I knew from that point on, this was something I wanted to do for the rest of my life. Since then, I've been fortunate enough to have the opportunity to learn and apply Python as an additional language in my arsenal.

If I'm not working on some kind of automation solution for a client or reading to learn more about tech, you'll most likely catch me playing some video games, watching anime, or just jamming on my guitar.
